# 🛡️ DeepShield 智能反诈预警系统

**DeepShield** 是您的专属数字安全卫士，它能 24 小时全天候守护您的收件箱，自动揪出隐藏在海量信息中的各类骗局。不管是不法分子的伪装邮件，还是危险的虚假短信，DeepShield 都能用最容易理解的方式告诉您：“这是什么骗局，您应该怎么办”。

---

## ✨ 为什么选择 DeepShield？

### 🧠 多个“AI 大脑”联合诊断
就像去医院看病会有专家会诊一样，DeepShield 不依赖单一的判断。它内置了多位“虚拟安全专家”，从文字模式、逻辑语境、诈骗套路等不同角度去分析每一封邮件。只有当专家们综合研判确认危险时，系统才会发出警报，极大地降低了误报率。

### 📝 听得懂的“人话”报告
告别生涩难懂的技术代码和警告提示！当我们抓出可疑邮件后，系统会自动为您生成一份非常直观的**反诈定性分析报告**。
报告会直接告诉您：
- 骗子想干什么（比如：兼职刷单、中奖通知、伪冒官方等）
- 邮件里哪里存在猫腻
- 您现在应该怎么处理以避免损失

### ⚙️ 真正解放双手的全自动监控
您只需要在系统里绑定一次您的邮箱（目前支持 163 邮箱），DeepShield 就会像一个不知疲倦的卫士，在后台默默地每 5 分钟帮您巡逻一次。一旦发现威胁，它会自动将危险记录在案，您随时可以登录控制面板查看，一切无需人工干预。

### 🎨 赏心悦目的使用体验
安全软件不应该只有冷冰冰的界面。我们为 DeepShield 设计了媲美大厂的现代化界面。无论是平滑的动画效果、清晰的数据面板，还是精心设计的图标，都旨在让您的每一次使用成为一种享受。

## 🛠️ 快速上手

只需要简单的几步，您的私人反诈卫士就能正式上岗：

### 第一步：启动系统
在您的电脑上双击启动程序（或在终端中运行 `python app.py`），稍等片刻，系统后台服务就会启动。
打开浏览器，访问 `http://127.0.0.1:5000`，您就能看到 DeepShield 的控制面板了。

### 第二步：配置您的私人大模型服务
1. 点击左侧菜单进入 **“系统设置”**。
2. 填入您的阿里云 DashScope 服务密钥（如果您想使用高级 AI 报告分析功能）。

### 第三步：让卫士开始工作
1. 点击左侧菜单的 **“邮箱管理”**。
2. 输入您的邮箱账号以及授权码（不是网页登录密码哦，是邮箱服务商提供的应用授权码）。
3. 绑定成功后，系统就会立刻开始第一次大扫除，并在之后保持自动巡逻！

## 🙋‍♀️ 常见问题解答 (FAQ)

**Q：系统会自动回复骗子或者删除我的邮件吗？**  
A：**绝对不会。** DeepShield 遵循“只读和预警”原则，它只会读取新邮件进行分析，并把有风险的邮件记录在系统中给您看，绝不会擅自删除或回复您的任何邮件，100% 保护您的数据安全。

**Q：为什么我感觉电脑有时候风扇转得比较快？**  
A：这是正常的现象。当 DeepShield 在进行深度扫描和 AI 交叉验证时，需要进行一些运算。扫描结束后就会恢复平静。

**Q：系统安全吗？我的邮件会被别人看到吗？**  
A：所有的分析和判断都是在您的个人电脑本地完成的，或者通过官方加密接口发送给大模型进行分析。我们不会在任何第三方服务器上存储您的私人邮件数据。
